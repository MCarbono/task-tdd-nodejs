## 📁 Project download

```bash
    #clone the repository
    $ git clone https://github.com/mcarbono/task-tdd-nodejs

    #Access the project folder
    $ cd task-tdd-nodejs

    #install dependencies
    $ npm i
```
---

## 📜 Configuration

- Set values to .env variables

## 🧪 Run tests
